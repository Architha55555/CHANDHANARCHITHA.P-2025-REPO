The XML Editor is the primary editor for authoring in Flare Desktop. It displays when you open a topic, snippet, or template page from the Content Explorer, and provides access to the underlying XML structure of documents in a comfortable visual authoring environment. This editor is used to enter, modify, and format the content that users see in the output. Although this editor lets you produce XML files, you do not need to know anything about XML to use it.
